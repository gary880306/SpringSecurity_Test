學習技術 : SpringSecurity 

專案需求 : 保護一個「電影網站」( 單一式 )

開發流程 : 會員功能實作 ( Authentication & Authorization ) ->  安全性保護 ( CORS & CSRF )

使用工具 : PostMan , Intellij

主要功能 : 

          1.使用 SpringSecurity 實作出帳號密碼登入功能
          
          2.套用 RBAC 權限模型，實現權限管理
          
          3.使用客製化 Filter
          
          4.使用 CORS 與 CSRF 增加網站安全性
          
          5.完整單元測試
